Metapost Tutorials

  • The MetaFont book by Knuth is the ultimate resource for learning MetaFont concepts. There are ebook versions available in internet, but they don't have illustrations. It is recommended to have a printed copy of this book.
  • John D. Hobby. A METAFONT-like System with PostScript Output. TUGboat, 10(4), 1989.
  • John D. Hobby. METAPOST — A User’s Manual, 2008.
  • Learning METAPOST by Doing
  • MetaPost for Beginners
  • The TUG page of metapost has a collection of examples and tutorials.

Colors

Please refer the documentation for mpcolornames package.

Debugging

Follow the messages in the console(displayed at bottom part of page) and use the line number to locate the error.

Examples

Reading many examples is a good way to learn the MetaPost concepts. You may also refer the works of user santhoshtr as starting point. Typefaces like Nupuram, Malini are designed using MetaPost, Please refer them for elaborate samples and documentation.
